À propos de notre projet

Découvrez les meilleurs restaurants de Lyon avec notre application. Une structure en arbre pour une navigation facile et une expérience culinaire exceptionnelle.

Technologies Utilisées

  • Flask
  • SQLite
  • HTML/CSS
  • JavaScript
  • Leaflet
  • Tailwind CSS

Pourquoi Flask ?

Flask a été choisi pour ce projet en raison de sa simplicité et de sa flexibilité. Il permet de créer rapidement des applications web tout en offrant la possibilité de s'étendre avec des extensions. De plus, Flask est bien adapté pour des projets de petite à moyenne taille, ce qui correspond aux besoins de ce projet.

Structures de Données

  • Noeud : Représente un noeud dans un arbre.
  • Pile : Implémentation d'une pile pour gérer les noeuds.
  • PileTree : Utilise une pile pour parcourir les arbres.
  • recursiveTree : Utilise des méthodes récursives pour parcourir les arbres.
